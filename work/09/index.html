IKEDA KOUICHI PORTFOLIO

平昌オリンピック

DESCRIPTION

Visual Design

フィギュアスケートの羽生結弦選手を始めとするメダル獲得が期待される日本人選手の情報を、どこよりも速く、詳しく、分かりやすく伝え、平昌オリンピックを盛り上げる

DESIGN CONCEPT

冬季オリンピックの「冬」を連想させる「雪、氷、結晶、空の青いグラデーション」を中心に表現する
競技毎にピクトグラムを作成し、競技の種別・特徴を視覚的に表現する
