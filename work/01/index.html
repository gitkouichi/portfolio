ICHIRO｢記憶｣と｢記録｣

DESCRIPTION

UI・UX Design / Visual Design / Cording

偉大な記録を射程圏に捉えたイチローの凄さをもう一度、世の中に伝える
野球のコア層を中心にするが、裾野は幅広くし、ライト層にも届ける

DESIGN CONCEPT

Simple・・・まっすぐ・芯の通ったカッコよさ

イチローの「記録と記憶」を、イチローのイメージ「シンプル、栄光、孤高」にあったデザインで野球ユーザーに届ける
イチローの凄さを「！」な体験で伝える（パララックスなど）

CODING TECHNIQUE

HTML / CSS / JavaScript / SVG

スクロール毎にコンテンツが切り替わる紙芝居風パララックス
初回読み込み時のページサイズを抑えるため、LOADING画面を設け、大判写真など必要なデータをダウンロードした後、メインコンテンツを展開
