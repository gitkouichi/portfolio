問トレアプリ

DESCRIPTION

UI・UX Design / Visual Design / Cording

医学生に向けて、いつでもどこでも、医師国家試験の対策ができる完全無料の問題演習アプリを目指す
国家試験の演習、学習進捗の確認、周囲の学習ペースを可視化することで、医学生の学習をサポートするアプリを開発する

DESIGN CONCEPT

Smart・・・賢明な医学生の生活や価値観にフィットするデザインを目指す。スタイリッシュで洗練された美しさを追求する
Graphical・・・医学生が無理なく、楽しく学びを続けられるデザインを目指す。明快に伝わる表現をする
Simple・・・規則性と一貫性を保ったデザインを目指す。無駄を削ぎ落とし、コミュニケーションをスムーズにする

CODING TECHNIQUE

HTML / CSS / JavaScript
